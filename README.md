STM Samples
===========

This code shows how to use [cpp_stm_free](https://github.com/graninas/cpp_stm_free) STM library.

Installation
------------

Run in the project dir:

```bash
git clone --branch v0.4 git@github.com:graninas/cpp_stm_free.git ./lib/cpp_stm
```

Then build with Qt.

Requirements
------------

- GCC 7.2
